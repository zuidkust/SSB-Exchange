# SSB 交易所 — 玩家手册

> 这是一份面向玩家的指南，帮助你理解游戏的各个环节。市场中的所有公司、股票、基金、期货、球队和新闻均为虚拟内容，仅供游戏体验。

---

## 1. 欢迎来到 SSB 交易所

SSB 交易所是一个模拟投资游戏。你将获得 **100 万** 虚拟启动资金，在虚构的市场中交易股票、基金和期货，还可以参与体育竞猜。

每个玩家使用分配的固定账号登录，首次登录时需要设置密码。未设置密码前，你的账号处于未激活状态，不会出现在排行榜等公开页面。

游戏按「期」推进，每一期代表一个小时的市场时间。

## 2. 市场与股票

### 交易时间

市场按北京时间运行。每日的 08:00 到 17:00 开放交易。

每到一个整点（08:00、09:00、10:00、11:00、13:00、14:00、15:00、16:00），市场自动推进一期，价格随之更新。中午 12:00 休市，不推进。

最后一期在 16:00 推进，但交易窗口会延续到 17:00，玩家有一个小时的反应时间。17:00 之后封盘，只能查看行情，不能交易。

### 股票概览

市场共有 **36 只** 虚构股票，分属六个板块：

**科技**
半导体、软件/云计算、通信、消费电子
典型股票：炬芯科技、达云数据、诺威、朗宗电子

**医药健康**
医药、医疗器械、生物科技
典型股票：济平医药、季骏基因、敏麟医疗、明洛医药

**消费**
食品饮料、零售、教育、传媒文娱
典型股票：文峰酒业、低能游戏社、万熹文娱、广实零售

**新能源与制造**
光伏储能、新能源车、高端制造
典型股票：曜琅光能、大米汽车、邓氪汽车、冠达精工

**金融地产**
金融、房地产
典型股票：永翔银行、立信保险、提树地产、璧岳置业

**周期与基础**
资源能源、建筑建材、农业、物流、环保
典型股票：广威矿业、白璞电力、迈达物流、善海环保

每只股票标注了风险等级：高风险股票波动剧烈，可能短期内大幅涨跌；低风险股票走势相对平稳；中风险介于两者之间。

### 价格波动受哪些因素影响？

股票价格不是凭空变化的，而是多个因素共同作用的结果：

- **新闻事件** — 公司公告、行业政策、市场传闻都会在数期内持续影响股票价格。重要新闻的影响通常更深远（详见新闻章节）。
- **板块联动** — 同一板块的股票走势常常同步。如果科技板块整体走强，板块内多数个股也会跟涨。
- **市场情绪** — 大盘整体的涨跌氛围会影响个股。强势市场中多数股票倾向上涨，弱势市场中则相反。
- **买卖力量** — 一只股票被大量净买入或净卖出时，会对价格形成轻微推动。
- **趋势延续** — 如果一只股票连续上涨或下跌，可能形成短期趋势，在一段时间内沿原方向继续运动。
- **随机波动** — 市场上永远有不可预测的因素，短期随机涨跌是正常现象。

### 交易规则

| 规则 | 说明 |
|------|------|
| 最小交易单位 | 1 手 = 100 股 |
| 手续费 | 买入和卖出各收 0.1% |
| T+1 | 当期买入的股票，下一期才能卖出 |
| 单只股票上限 | 持有的单只股票市值不能超过你的总净资产 |
| 涨跌停 | 单期最大涨跌幅为 ±10% |
| 涨停时买入 | 买盘拥挤，最多只能买入正常限量的一半；但可以正常卖出 |
| 跌停时卖出 | 卖盘拥挤，最多只能卖出正常限量的一半；但可以正常买入 |

### 买卖操作

在市场页面点击任意股票行，会打开详情弹窗：

- 左侧是该股票的基本信息、价格走势图和关联新闻
- 右侧是交易面板。你可以使用快速按钮选择手数或金额，也可以自定义输入。确认后系统会实时反馈成交结果

## 3. 基金

### 什么是基金？

基金是一种集合投资工具。你申购基金份额，基金按规则持有一篮子资产，净值随底层资产表现变动。相比单只股票，基金通过分散持仓降低了个别股票的风险。

市场共有 **11 只** 基金，分为三大类：

### 被动指数型基金

这类基金按固定规则配置股票，不涉及基金经理的主动判断：

- **山海蓝筹精选 (SH01)** — 持有所有低风险股票，等权重配置
- **广迪 20 指数 (GD01)** — 按价格加权持有 20 只精选股票

### 主动管理型基金

这类基金由基金经理主动调仓，每期根据市场情况调整持仓。要注意：**主动基金的持仓展示会延迟一期**——你能看到前一期持有的股票，但看不到当期新调整的持仓。

五位经理各有一套投资偏好：

- **天一成长混合 (TY01)**｜经理：施天 — 偏好近期涨势强劲的科技股
- **天一健康优选混合 (TY02)**｜经理：杜智桢 — 偏好被低估的医药股
- **山海金融地产混合 (SH02)**｜经理：葛正衡 — 做空近期涨幅过大的金融地产股
- **迪必消费优选混合 (DB01)**｜经理：林勋达 — 均衡配置消费板块
- **迪必新能动量混合 (DB02)**｜经理：康明弘 — 追随新能源与制造板块的趋势

### 独立资产基金

这类基金不跟踪本地股票，净值独立运行：

- **天一全球精选 (TY03)** — 模拟海外市场资产，与本地股市关联度低
- **山海黄金 ETF (SH03)** — 跟踪黄金资产，与股市的关系会根据市场阶段切换（避险反向、风险同向或独立震荡）
- **广迪现金宝 (GD02)** — 货币类资产，净值缓慢稳定增长，波动极小
- **广迪纯债 (GD03)** — 债券类资产，票息缓慢累积，过程中伴随利率波动带来的涨跌起伏

### 申购与赎回

| 规则 | 说明 |
|------|------|
| 申购费率 | 0.1%（广迪现金宝免申购费） |
| 赎回费率 | 0.1%（广迪现金宝免赎回费） |
| T+1 赎回 | 当期申购的基金份额，下一期才能赎回 |
| T+0 赎回 | 广迪现金宝申购后当期即可赎回 |
| 最小金额 | 申购金额无最低限制，赎回份额无最低限制 |

基金详情页左侧列出所有基金，点击选择后在右侧查看该基金的净值走势图、持仓成分和交易操作区。

## 4. 期货

期货交易是一种高风险高收益的投资方式。你需要缴纳保证金来开仓，杠杆会成倍放大你的盈亏。

### 期货品种

目前共有 10 个期货品种，分属四类：

| 类别 | 品种 | 每手合约价值 | 最高杠杆 |
|------|------|------------|---------|
| 商品 | 原油、铜、锂、大豆、黄金 | 不同规格 | 8-10 倍 |
| 指数 | 天际 100、AC500 | 不同规格 | 8-12 倍 |
| 虚拟币 | S 币、花剑币 | 不同规格 | 8 倍 |
| 外汇 | Champion Dollar | 不同规格 | 12 倍 |

### 杠杆是什么意思？

杠杆让你用少量资金撬动大额交易。举个例子：如果你用 **5 倍杠杆** 开仓 1 张铜合约（每张保证金 1,500 元），你实际只需要 1,500 元就能持有价值 7,500 元的合约。

如果铜价上涨 10%，你的合约价值增加 750 元——相对于你 1,500 元的保证金，盈亏是 **+50%**。但如果铜价下跌 10%，亏损也是 **-50%**。杠杆越高，盈亏放大越剧烈。

### 做多与做空

- **做多** — 你预期价格会涨。价格上涨时盈利，价格下跌时亏损。
- **做空** — 你预期价格会跌。价格下跌时盈利，价格上涨时亏损。

### 保证金和爆仓

开仓时你需要缴纳保证金。如果价格朝不利方向大幅移动，保证金不足以覆盖亏损，就会触发**强制平仓**（爆仓）。

每张持仓卡片上会显示：
- **爆仓价** — 价格触及此位时自动平仓
- **爆仓距离** — 当前价格距离爆仓价还有多远

### 强平（爆仓）的顺序

如果你的期货爆仓后保证金仍不足以覆盖亏损，系统会按以下顺序自动卖出其他资产来弥补亏空：

1. 先平掉所有期货持仓
2. 卖出股票（高风险 → 中风险 → 低风险）
3. 卖出基金（高风险 → 中风险 → 低风险）
4. 最后卖出债券基金和货币基金

强平卖出绕开 T+1 限制且免手续费。

### 开仓额度限制

你可以开多少期货仓位取决于两点：
- 你的可用现金
- 你的期货风险敞口上限（总资产的 80%，各品种分类有子限额）

## 5. 赛事竞猜

### SBA 篮球联赛

SBA 是一个虚构的篮球联赛，共 **16 支** 球队，每队拥有各自的首发和替补阵容。

每两周开启一个新赛季。赛季包括：
- **常规赛** — 30 轮双循环，每支球队与其他 15 队各打两场（主客场各一）
- **季后赛** — 常规赛排名前 8 的球队进入，依次打四分之一决赛 → 半决赛 → 总决赛

### 比赛胜负如何决定？

每场比赛的胜负取决于几个因素：

1. **球队实力** — 由首发和替补球员的能力综合决定。首发球员贡献更大权重
2. **近期状态** — 近几场比赛的净胜分高低会影响球队的势头
3. **主场优势** — 主场球队有一定程度的获胜加成
4. **随机性** — 比赛中永远存在无法预测的因素，强队也可能翻车

最终的实际比分在胜负结果确定后，会根据预期分差和比赛随机性模拟生成。

### 战力星级

每支球队标注了战力星级（1-5 ★）。星级只表达**当前阵容的基础实力**，由首发和替补球员能力计算，并会在球员交易、休赛期调整或选秀成长后立即变化。

| 星级 | 当前阵容基础实力 |
|------|------------------|
| 5★ | ≥ 80 |
| 4★ | ≥ 76 |
| 3★ | ≥ 72 |
| 2★ | ≥ 68 |
| 1★ | < 68 |

4★和5★代表常规赛前八的高可靠梯队，但不是保送资格，也不代表单场必胜。投注时应同时参考卡片展示的战绩与近期状态；赔率还会计入近期状态和主场优势。

### 球员流动

每个赛季期间和结束后，球员会在球队之间流动：
- 常规赛进行中的**球员交易**（部分球员轮换到新球队）
- 赛季结束后的**休赛期调整**（更大规模的球队阵容变更）
- 常规赛排名靠后的弱队获得**选秀补强**机会（概率递减，但补强幅度可观）

这些机制使联盟格局不会长期固化，强弱队的位置会随时间推移发生变化。

### 竞猜规则

赛事竞猜包含两条互不冲突的玩法：

- **单场胜负竞猜** — 每场比赛开放固定的主场胜 / 客场胜两个选项
- **季后赛系列赛胜者竞猜** — 对阵确定后，可在季后赛树状图中竞猜整个系列赛的最终晋级队伍；G1 开始后停止投注

点击赔率后，系统会先弹出确认窗口，显示竞猜类型、选择球队、金额、锁定赔率和预计返还。确认后才会正式提交；若提交失败，页面会使用弹窗说明具体原因。

| 规则 | 说明 |
|------|------|
| 单笔最低 | 1,000 |
| 单场累计上限 | 100,000（同场所有投注合计） |
| 单系列赛累计上限 | 200,000（同系列赛所有投注合计） |
| 锁定赔率 | 下注时的赔率即为最终赔率，不受后续赔率变化影响 |
| 扣款时机 | 下注时立即从账户现金中扣除本金 |
| 获胜派奖 | 单场或系列赛结算后自动到账：返还奖金 = 本金 × 锁定赔率 |
| 取消与作废 | 比赛取消、系列赛市场作废或赛季作废时，未结算本金按原额退还 |

赛事首页会显示可用资金、待开奖本金和本赛季盈亏。单场比赛投注继续在赛事首页展示；系列赛胜者投注位于季后赛树状图，两种投注可以同时参与。

赔率较低的一方是更被看好的球队。单场比赛开始后不能继续投注该场比赛；系列赛 G1 开始后不能继续投注该系列赛。

### 什么是赔率？

赔率 2.0 意味着：如果你下注 1,000 并猜中，你将获得 2,000（净赚 1,000）。赔率 1.5 意味着下注 1,000 猜中后获得 1,500（净赚 500）。赔率反映了市场对胜负方向的判断，但不是绝对的——被看好的球队也可能输。

## 6. 新闻

### 新闻的作用

新闻是市场中最重要的信息源。每期推进后都会产生新的新闻，这些新闻会影响股票、行业乃至整个市场的价格走势。

### 新闻类型

新闻按来源分为多种类型，在页面上通过不同的标签区分：
- **公司公告** / **行业快讯** — 来自公司和行业的正式信息
- **政策解读** / **政府通报** — 宏观层面的政策信息
- **市场传闻** / **花边新闻** — 市场中的各种传闻和八卦

### 真实与虚假

新闻不等于事实。有些新闻是可信的，有些只是市场传闻。在阅读新闻时需要自己判断信息的真实性。

当虚假传闻被揭穿后，会出现**辟谣或反转报道**。被辟谣的新闻对价格的影响会消失，甚至产生反向冲击。

重要规则：
- 越权威的新闻来源（如官方公告），信息可信度通常越高
- 新闻对价格的影响会在数期内逐渐释放，不会一期就结束
- 同一只股票如果同时有多条新闻在影响它，作用会相互叠加或抵消

### KOL 评论

新闻页面右侧有 **KOL（意见领袖）** 评论栏。这些是模拟市场中的财经自媒体评论者，每期会从评论者池中随机选取若干位，对当前新闻事件发表评论。

- 评论者分为三类：**执业分析师**（专业背景，带蓝色徽章）和**财经领域创作者**（自媒体背景）和**网友**
- KOL 的评论可能引用某条新闻（如"#新闻1"），点击引用链接可以跳转到对应新闻卡片
- KOL 评论仅供参考和娱乐——他们的判断不一定正确，不要盲目跟从

### 新闻阅读建议

每期的新闻会优先显示官方澄清、政策类信息，其他新闻按时间排序。建议优先关注：
1. 是否有与你持仓股票直接相关的新闻
2. 行业层面的政策变动（会影响整个板块）
3. 宏观层面的市场情绪变化

## 7. 持仓

持仓一览页面向你展示所有已激活玩家的资产配置状况。

### 分段条形图

每位玩家一条水平色条，色条按股票、基金、期货三种资产类别分段。一个色段越长，说明该玩家在这个类别上的投入比例越高。

### 悬停查看明细

将鼠标悬停在任意色段上，会弹出弹窗显示该玩家的具体持仓明细：
- 持有的每项资产名称和当前市值
- 各项资产在当期的涨跌幅
- 涨显示绿色，跌显示红色

### 净资产

页面上显示的净资产计算方式：

> 净资产 = 现金 + 股票市值 + 基金市值 + 期货市值 − 贷款负债

这是衡量你综合实力的核心指标。

## 8. 贷款

贷款页面包含两个标签：**银行贷款** 和 **个人借贷**。

### 银行贷款

#### 信用评级

每个玩家有信用星级（★ 一级到三级），初始为一星。星级越高，可贷额度和可选期限越多：

| 信用等级 | 最高可贷比例（净资产） | 可选期限 |
|---------|-------------------|---------|
| 一星 | 50% | 短 |
| 二星 | 60% | 短、中短 |
| 三星 | 70% | 短、中短、中长 |

#### 如何提升信用星级？

满足以下两个条件并按时还清贷款，即可提升一级（最高三星）：

1. **贷款额 ≥ 10 万** — 低于 10 万的贷款不计入升级资格
2. **持满 1 个自然日** — 贷款必须持有至少一整天（即经历一个完整的交易日循环），当天下款当天就还清不计入

每还清一笔满足上述条件的贷款，星级 +1。两笔合格还款即可从一星升到三星。

#### 利率

银行贷款采用**阶梯利率**：贷款额的前一部分按较低利率计息，超出部分按较高利率。具体来说，贷款额越高，综合资金成本越高。

#### 还款与逾期

- 每期自动计提利息
- 到期前会有弹窗提醒
- 逾期不还将触发**强制平仓**：系统自动卖出你的持仓资产来还债，顺序与期货强平相同

### 个人借贷（P2P）

个人借贷是玩家之间的借贷市场。

#### 借用

如果你需要借入资金，可以在市场上发布「借用」挂单，设定想要的利率档位（四档可选，利率越高越容易匹配到出借人）和期限。

#### 出借

如果你有闲置资金，可以发布「出借」挂单。出借资金会在借款期限内产生利息收入，到期后收回本金。

需要注意：如果借款人逾期无法还款，你可能损失本金。出借人的待收贷款会计入你的净资产，一旦借款人破产，这部分资产将减值。

#### 智能匹配

系统会根据利率和期限偏好，自动推荐与你最匹配的对手挂单。发布挂单后也可以浏览市场中的其他挂单，手动选择匹配。

#### 个人借贷参数

| 规则 | 说明 |
|------|------|
| 单笔金额 | 1 万 ~ 20 万 |
| 利率档位 | 四档，从低到高 |
| 期限档位 | 四档，从短到长 |
| 逾期处理 | 逾期触发强平（银行贷款优先于民间贷款） |

## 9. 排行榜

排行榜页面展示两个维度的排名：

### 净资产排名

按每个玩家的净资产从高到低排序。这是衡量综合财富的最主要指标。

### 当日收益排名

按每个玩家当天的收益比例排序。这反映了短期的投资表现，即使总资产不高，当天的操作也可能让你冲上日榜。

你的当前排名会在排名行中高亮显示。

## 10. 个人账户

账户页面是你的个人资产仪表盘，汇总了你所有的资金和持仓状况。

### 资产总览

页面顶部的资产条显示：
- **可用现金** — 当前可以用于交易的现金余额
- **股票市值** — 所有股票持仓的当前市场价值
- **基金市值** — 所有基金持仓的当前净值总和
- **期货市值** — 所有期货持仓的当前价值
- **净资产** — 综合上述所有项目的净值

### 贷款提醒

如果你的银行贷款或个人借贷即将到期，此卡片会显示当前应还金额和剩余期限。

### 赛事竞猜

显示当前待开奖的竞猜本金、赛季累计盈亏，以及近期的竞猜记录。

### 持仓卡片

分三张卡片展示你的持仓详情：
- **股票持仓** — 每只股票的持股数、当前市值、盈亏状况
- **基金持仓** — 每只基金的份额数、当前净值、盈亏状况
- **期货持仓** — 每张期货合约的方向、张数、杠杆倍数、盈亏状况

### 交易记录

底部折叠面板展示你最近 15 条股票、基金和期货的交易记录，按期数倒序排列。
