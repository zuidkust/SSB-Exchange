# SSB 交易所 新手指南

欢迎来到 SSB 交易所！以下是各个板块的使用指南，帮助你快速上手。

## 交易板块（核心）

- [股票交易](./market.md) - 买卖 36 只不同行业的股票
- [基金投资](./funds.md) - 申购赎回 11 只不同类型的基金
- [期货交易](./futures.md) - 高杠杆期货合约，支持做多/做空

## 功能板块

- [新闻资讯](./news.md) - 查看市场新闻和 KOL 评论
- [持仓一览](./holdings.md) - 查看所有玩家的持仓情况
- [我的账户](./account.md) - 查看资产总览和交易记录
- [排行榜](./ranking.md) - 查看净资产和收益率排名
- [贷款服务](./loan.md) - 借款投资，有利息和还款期限

## 管理板块

- [运营台](./admin.md) - 管理员专用后台（仅管理员可见）

---

## 新手入门建议

1. **先看新闻**：了解当前市场动态
2. **浏览行情**：查看股票价格和涨跌
3. **小额试水**：先买入 1-2 手股票熟悉流程
4. **分散投资**：不要把所有资金投入一只股票
5. **关注账户**：定期查看净资产变化

## 风险提示

- ⚠️ 股票可能亏损，期货风险更高
- ⚠️ 贷款需要按时还款，否则可能破产
- ⚠️ 新闻可能有假消息，注意辨别
