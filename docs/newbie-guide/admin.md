# 运营台 新手指南

## 快速了解
- 这是什么：管理员专用后台，控制整个交易所的运行
- 核心功能：管理股票和玩家，控制市场节奏

## 你应该做什么
1. 先查看"账号状态"，了解当前玩家情况
2. 使用"邀请码管理"为新玩家生成注册码
3. 定期检查"最近交易"，监控市场异常

## 你可以做什么
- **股票管理**：新增或编辑股票，调整初始价格和风险等级
- **邀请码管理**：批量生成邀请码，设置玩家昵称，撤销未使用的码
- **账号管理**：查看玩家资产、登录和交易情况
- **市场控制**：手动推进期数、临时解除封盘、重置整个市场周期
- **玩家管理**：重置单个玩家密码，或重置破产玩家账户（复活）

## 风险与后果
- ⚠️ **重置市场周期**：清空所有行情、持仓、交易记录、新闻和排行榜，账号密码保留
- ⚠️ **重置所有玩家密码**：所有普通玩家回到未激活状态，需要重新设置密码
- ⚠️ **重置玩家账户**：清空贷款、持仓、交易记录，现金恢复100万，密码重置

## 注意事项
- 仅管理员可见和操作，普通玩家看不到运营台
- 封盘状态下玩家无法交易，但可以查看行情
- 手动推进只用于自动时钟异常或临时修复
- 市场休眠后需要先"恢复本局"才能继续交易和推进
