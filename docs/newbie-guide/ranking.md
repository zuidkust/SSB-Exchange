# 排行榜 新手指南

## 快速了解
- 这是什么：查看所有玩家资产和收益排名的实时榜单
- 核心功能：对比净资产排名、对比今日收益率排名、观察其他玩家表现

## 你应该做什么
1. 进入「排行榜」页面，查看当前期数和自己的排名
2. 对比净资产排行榜，了解自己的资产水平
3. 对比今日收益率排行榜，了解自己的投资表现
4. 观察排名靠前玩家的策略，学习投资思路

## 你可以做什么
- **查看净资产排名**：显示所有玩家的总资产（现金+股票+基金+期货-贷款）从高到低排序
- **查看今日收益率排名**：显示所有玩家的今日收益百分比从高到低排序
- **查看个人排名**：顶部显示你在两个榜单中的具体名次
- **观察其他玩家**：查看其他玩家的昵称和资产/收益数据

## 风险与后果
- ⚠️ **排名实时变化**：每期（每小时）更新一次，排名会随市场波动而变化
- ⚠️ **只显示活跃玩家**：未激活或已破产的玩家不会出现在排行榜中
- ⚠️ **净资产包含负债**：净资产 = 总资产 - 贷款，有贷款的玩家净资产会降低

## 注意事项
- 排行榜每天更新8期，对应交易时间8:00-16:00（北京时间）
- 今日收益率以当天开盘时的资产为基准计算
- 排名只统计已激活且未破产的玩家
- 排行榜数据来自实时计算，与持仓页面数据一致
