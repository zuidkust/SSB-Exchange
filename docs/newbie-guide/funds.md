# 基金投资 新手指南

## 快速了解
- 这是什么：通过购买基金份额，间接投资股票、债券、黄金等资产，无需自己选股
- 核心功能：分散投资降低风险、专业基金经理管理、一键申购赎回

## 你应该做什么
1. 了解基金类型：行业基金（跟随股票市场）和独立资产（黄金、货币、债券等）
2. 根据风险偏好选择：低风险（蓝筹、货币、债券）、中风险（指数、消费、金融地产）、高风险（科技、新能源、海外）
3. 输入申购金额，系统自动扣除手续费并计算份额
4. 赎回时输入份额，资金到账后可用于其他投资

## 你可以做什么
- **申购基金**：输入金额（最低1元），按当前净值计算份额，手续费0.1%
- **赎回基金**：输入份额，按净值计算金额，手续费0.1%
- **查看持仓**：实时显示持有金额、盈亏情况
- **查看基金详情**：净值走势图、持仓成分、基金经理信息
- **查看交易记录**：历史申购赎回记录

## 风险与后果
- ⚠️ 净值波动：基金净值会涨跌，可能亏损本金
- ⚠️ 赎回延迟：除现金宝外，申购份额需T+1日才能赎回
- ⚠️ 强制清算：破产时系统会自动赎回基金用于还款
- ⚠️ 手续费：每次申购赎回都收取0.1%手续费（现金宝除外）

## 注意事项
- 现金宝（GD02）：免手续费，可T+0赎回，适合存放闲置资金
- 主动管理基金：有基金经理操盘，收取0.04%/期管理费，持仓会动态调整
- 被动管理基金：无管理费，持仓固定或按规则调整
- 风险等级标签：低风险（绿色）、中风险（黄色）、高风险（红色）
- 基金经理风格：动量（追涨）、价值（抄底）、逆向（反向操作）、平衡（稳健）、趋势（跟随）
